# Project Vision & Goals

## Project summary
Safora is an educational financial planning platform for Massachusetts families. It offers class registration (via JanaGana CRM), portfolio balance tools, and a life insurance recommendation calculator — all with regulatory compliance built in.

## Primary goals
1. **Class sign-ups** — Visitors register for upcoming financial planning workshops (Zoom and in-person) through JanaGana portal integration.
2. **Planning tools** — Free educational calculators (portfolio balance, life insurance) that run client-side without storing personal data.
3. **Regulatory compliance** — Clear disclaimers for insurance and financial advisory content; no personalized investment advice.

## CRM integration
Class registration, event listings, and lead capture are handled by [JanaGana](https://janagana.namasteneedham.com) (`namaste-boston` tenant). This site embeds events and links to the visitor portal.

## Success metrics
- Class registration conversion
- Tool engagement (balance planner, life insurance)
- Compliance review pass rate
- Test and deployment stability
