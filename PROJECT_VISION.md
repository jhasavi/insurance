# Project Vision & Goals

## Project summary
This project provides an insurance quoting and comparison platform built with Next.js and Prisma. It aims to help users obtain quotes across carriers, compare options, and submit applications.

## Vision
To build a reliable, compliant, and delightful digital insurance experience that simplifies shopping for coverage, increases conversion for carriers, and scales to support multiple product lines.

## Short-term goals (next 3-6 months)
- Stabilize core flows: quote intake, comparison, and submission.
- Ensure CI/CD reliability on Vercel and fix flaky deployments.
- Improve test coverage for critical user journeys (Playwright E2E).
- Harden observability: Sentry, logging, and error handling.

## Long-term goals (6-24 months)
- Add multi-carrier policy issuance and endorsements.
- Expand product catalog (additional insurance lines).
- Implement analytics-driven personalization and carrier insights.
- Optimize for enterprise integrations (APIs, webhooks, partner portals).

## Success metrics
- Conversion rate on quote submissions
- Time to first quote (performance metrics)
- Test pass rate and deployment stability
- Customer satisfaction and retention

## Stakeholders
- Product, Engineering, Legal/Compliance, Carriers, and Ops.
